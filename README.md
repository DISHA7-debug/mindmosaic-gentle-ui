# MindMosaic

MindMosaic is a frontend web application built to provide a clean, intuitive, and responsive user interface for organizing content and navigating different sections smoothly.

## Project Overview

This project is developed using modern frontend tools and follows a component-based architecture for better scalability and maintainability.

## Tech Stack

The project is built with:

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- React Router
- TanStack React Query

## Getting Started

Follow the steps below to run the project locally.

### Prerequisites

Make sure you have the following installed:

- Node.js (v18 or later recommended)
- npm

You can install Node.js using nvm if required:  
https://github.com/nvm-sh/nvm

### Installation

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Install dependencies
npm install
